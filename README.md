# Lubuk Linggau Smartcity 

## Fitur
- Login
- Berita
- Layanan Publik
- Agenda
- Data Sektoral
- Pengaduan

## Hak Akses
- Administrator
- Admin OPD

## Konfigurasi

* Clone project
* install dependecies
```bash
yarn install
```
* Sesuaiakan url pada ```src/constants/Url.js``` dengan url microservice
* Jalankan aplikasi
```bash
yarn start
```
